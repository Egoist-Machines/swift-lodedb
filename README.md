# LodeDBCore (Swift)

On-device vector, text, and hybrid search for macOS and iOS, over LodeDB's native
Rust core. This is the published SwiftPM distribution: a thin package whose binary
target is the `LodeDBCoreFFI.xcframework` from the corresponding LodeDB release.

```swift
.package(url: "https://github.com/Egoist-Machines/swift-lodedb", from: "2.0.0")
```

Sources, tests, issues, and docs live in the main repository:
https://github.com/Egoist-Machines/LodeDB (see `swift/LodeDBCore/README.md` and
`docs/swift-agent-contract.md`). This package is generated from LodeDB 2.0.0;
do not edit it here.
